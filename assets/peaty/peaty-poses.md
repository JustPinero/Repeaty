# Peaty the Parrot — Pose Reference for Leonardo.ai

Mascot for Repeaty (language learning PWA). Use the same base style across all poses for consistency: charming cartoon parrot, vibrant green and yellow plumage with teal accents, large expressive eyes, small friendly beak, clean modern mascot design, smooth vector-style shading, soft pastel background, Duolingo-adjacent but distinct.

**Tip:** Generate the portrait first, then use Leonardo's Image Guidance / Character Reference with that image for every pose below to keep Peaty consistent.

**Shared negative prompt:** realistic photo, scary, aggressive, dark, gritty, blurry, low quality, extra limbs, deformed beak, watermark, text, signature, cluttered background, multiple parrots

Peaty centered front-facing portrait, friendly closed-mouth smile, symmetrical composition, slightly larger head proportion for icon readability, simple solid background or transparent, optimized for small sizes (export 192px and 512px), no text or extra elements

Peaty waving with subtle motion, centered composition, large negative space around him for loading spinner overlay, soft brand-accent background, app launch feel

## Suggested Leonardo Settings for All Poses
- Model: Leonardo Phoenix or Leonardo Diffusion XL
- Style: Illustration or 3D Render (pick one and stick with it across all poses)
- Aspect ratio: 1:1 for icons/avatars, 3:4 for full-body in-app illustrations
- Guidance scale: 7
- Alchemy: on, PhotoReal: off
- Use Image Guidance with your finalized portrait at strength ~0.5 for character consistency
- Generate 4 per pose, pick the best, then upscale
