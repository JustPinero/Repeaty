# Request 1.1 — Monorepo Scaffold

## Objective
Stand up the working monorepo: pnpm workspaces, Vite + React + TS app, shared package, Supabase local stack, Tailwind + shadcn baseline, Vitest, Playwright, ESLint with `jsx-a11y`, Prettier. After this request, `pnpm install && pnpm --filter @repeaty/web dev` opens a "Hello, Repeaty" page and `bash scripts/validate.sh` passes green.

## RED Phase (write these tests FIRST)

1. `apps/web/src/test/shared-import.test.ts` — imports `{ version }` from `@repeaty/shared`, asserts it's a non-empty string.
2. `apps/web/src/env.test.ts` — calls the env validator with an empty `import.meta.env`, expects it to throw with a message naming `VITE_SUPABASE_URL`.
3. `apps/web/src/App.test.tsx` — renders `<App />` and asserts the heading text "Repeat after Peaty." is in the document with role `heading` (a11y query, not text).
4. ESLint config check: a deliberate `<div onClick={...}>` (no `role`/`tabIndex`/keyboard handler) in a fixture file fails `pnpm lint`.
