# Request 1.4 — Onboarding Flow

## Objective
After first login, send the user through a 3-step onboarding: (1) display name, (2) native language, (3) at least one target language with CEFR level. Completing onboarding writes `profiles.display_name`, `profiles.native_language_code`, and one or more `user_languages` rows. The user lands on `/app` (the dashboard placeholder; real content in 1.5). Re-entering `/app` after onboarding skips the wizard.
