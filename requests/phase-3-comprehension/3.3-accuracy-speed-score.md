# Request 3.3 — Accuracy × Speed Score Formula

## Objective
Define the formula that combines `similarity` (from 3.1) and `response_ms` into a single 0–100 score, ship it as a pure helper in `@repeaty/shared`, and wire it into the comprehension session's per-card result + completion summary. The formula must feel fair across very-easy and very-hard cards, and must not punish careful learners (no hard timeout — the curve flattens after a "thinking" budget).

## Acceptance Criteria → Test Mapping

| Criterion                                                                                  | Test                                                                       |
| ------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------- |
| Perfect accuracy (1.0) + 1s response → score ≥ 95                                          | `comprehension-score.test.ts > fast & correct`                             |
| Perfect accuracy + 30s response → score ≥ 60 (slow but not punished into the floor)        | `> slow & correct`                                                         |
| 0.0 accuracy → score = 0 regardless of speed                                              | `> wrong is wrong`                                                         |
| 0.7 accuracy + 5s → score in [40, 70] (graded partial credit)                              | `> partial credit`                                                         |
| Two consecutive 1s/1.0 results produce the same score (pure)                              | `> deterministic`                                                          |
| Score is clamped to [0, 100]                                                                | `> bounds`                                                                 |
| `bucket(score)` returns one of `'perfect' \| 'close' \| 'miss'` consistently               | `> bucket boundaries`                                                      |

## Proposed formula

```ts
// packages/shared/src/comprehension-score.ts
export function comprehensionScore(similarity: number, response_ms: number): number {
  if (similarity <= 0) return 0;
  // Speed factor: 1.0 at ≤ 2s, decays linearly to 0.5 at 30s, floor at 0.5.
  const speedFactor = response_ms <= 2_000
    ? 1
    : Math.max(0.5, 1 - ((response_ms - 2_000) / 28_000) * 0.5);
  return Math.round(Math.min(100, similarity * speedFactor * 100));
}

export type ScoreBucket = 'perfect' | 'close' | 'miss';
export function bucket(score: number): ScoreBucket {
  if (score >= 90) return 'perfect';
  if (score >= 60) return 'close';
  return 'miss';
}
```
