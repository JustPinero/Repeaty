# Request 3.5 — Stubbed AI Feedback Hook (real call lands in Phase 5)

## Objective
Add a `useFeedback({ kind: 'comprehension', ... })` hook that the comprehension result UI calls when the bucket is `'close'` or `'miss'`. In v1 the hook returns a small set of canned sentences (level-appropriate, native-language-aware). In Phase 5 the same hook is rewired to call the `generate-feedback` Edge Function (Claude proxy) — the public API stays stable, only the body swaps.

## Acceptance Criteria → Test Mapping

| Criterion                                                                       | Test                                                                       |
| ------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| Hook returns canned text for `bucket === 'close'`                              | `useFeedback.test.ts`                                                      |
| Hook returns gentler canned text for `bucket === 'miss'`                       | same                                                                       |
| Hook returns null for `'perfect'` (no feedback needed)                         | same                                                                       |
| Canned text references the user's native language code so localization is plumbed already | same                                                            |
| Hook is exposed via `@/features/feedback` (own surface, not buried in comprehension) | barrel + import test                                                  |
| ComprehensionSessionPage shows the feedback under the result when bucket ≠ 'perfect' | `ComprehensionSessionPage.test.tsx`                                  |

## Proposed shape

```ts
type FeedbackInput =
  | { kind: 'comprehension'; bucket: ScoreBucket; targetText: string; nativeText: string; userResponse: string; nativeLanguageCode: string };

export function useFeedback(input: FeedbackInput): { text: string | null; isLoading: boolean };
```

v1 implementation: pure function with a small lookup table keyed on bucket + nativeLanguageCode. No fetch, no async. Phase 5 swaps the body for an Edge Function call.
