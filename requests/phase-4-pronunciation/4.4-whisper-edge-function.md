# Request 4.4 — `score-pronunciation` Edge Function (Whisper)

## Objective
First Edge Function. Authenticated client posts `{ card_id, audio_storage_path }`; the function downloads the audio from Storage, calls OpenAI Whisper with the card's `language_code`, computes the similarity score using the existing `@repeaty/shared/similarity` helper, inserts a `pronunciation_attempts` row, and returns `{ attempt_id, whisper_transcript, similarity_score, expected }`. Wire contract documented in `references/api-contracts.md`.

## Acceptance Criteria → Test Mapping

| Criterion                                                                       | Test                                                                       |
| ------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| Returns 401 when JWT missing / invalid                                          | `score-pronunciation.test.ts` (Deno test)                                  |
| Returns 400 when body fails Zod parse                                           | same                                                                       |
| Returns 404 when `card_id` doesn't exist or isn't visible to the caller         | same                                                                       |
| Returns 403 when `audio_storage_path` doesn't start with the caller's user_id   | same                                                                       |
| Calls OpenAI Whisper with `language: card.language_code` and 15s AbortController | same                                                                      |
| On timeout: returns 504 `UPSTREAM_TIMEOUT`                                      | same                                                                       |
| On Whisper error: returns 502 `UPSTREAM_FAILED` with the original logged        | same                                                                       |
| Similarity score uses `@repeaty/shared/similarity` with the card's lang         | same                                                                       |
| Inserts a `pronunciation_attempts` row and returns `attempt_id`                  | integration test against live supabase                                     |
| Logs one structured JSON line per invocation per `references/api-contracts.md`  | manual / log inspection                                                    |

## RED Phase

Edge Function tests run via `deno test` against the function's `index.ts`. Set up under `supabase/functions/score-pronunciation/`. Mock OpenAI fetch + supabase admin client.

## GREEN Phase

1. **Function entry** `supabase/functions/score-pronunciation/index.ts`:
   - Zod parse the body.
   - `supabase.auth.getUser(token)` → user_id; 401 on failure.
   - SELECT the card via the user-context client (RLS-respecting); 404 if invisible.
   - Verify `audio_storage_path.startsWith(\`${user_id}/\`)`; 403 otherwise.
   - Service-role client downloads the audio.
   - POST to `https://api.openai.com/v1/audio/transcriptions` with `model: whisper-1`, `language: card.language_code`, 15s AbortController.
   - Compute similarity via `@repeaty/shared` (port to Deno-friendly path; `npm:` specifier or vendored).
   - Insert `pronunciation_attempts` row.
   - Log one JSON line.
2. **Shared error code enum** at `packages/shared/src/edge-errors.ts` (per api-contracts.md): `INVALID_PAYLOAD | UNAUTHENTICATED | FORBIDDEN_TIER | NOT_FOUND | RATE_LIMITED | UPSTREAM_TIMEOUT | UPSTREAM_FAILED | INTERNAL`.
