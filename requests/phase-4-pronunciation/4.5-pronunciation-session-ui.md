# Request 4.5 — Pronunciation Session UI

## Objective
A pronunciation session at `/app/decks/:deckId/pronunciation`. User sees the target word, taps Record, speaks, taps Stop. The Blob is uploaded to Storage; the Edge Function scores it; the user sees the Whisper transcript + similarity score + their own playback. After N cards: completion summary.
