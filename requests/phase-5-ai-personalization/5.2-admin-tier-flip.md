# Request 5.2 ŌĆö `/admin` Route + Tier Flip

## Objective
A minimal in-app admin surface for the Stripe-deferred-per-DEBT-001 reality. Only `profiles.is_admin = true` users can reach `/admin`. From there: list recent profiles, flip any user's `tier` between `free | pro | admin`. Each flip writes a `tier_change_log` row (5.1).
