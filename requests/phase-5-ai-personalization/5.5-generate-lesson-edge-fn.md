# Request 5.5 — `generate-lesson` Edge Function (Claude proxy)

## Objective
Third Edge Function. Pro/admin-only. Caller posts `{ language_code, topic_hint?, card_count? }`. The function loads the user's `cefr_level`, recent weak words from `reviews + comprehension_attempts + pronunciation_attempts`, builds a Claude prompt with a structured JSON output schema, parses + Zod-validates the response, and inserts a new deck (`source = 'ai_generated'`, `owner_id = caller`) plus its cards. Daily cap of 10 lessons/user via `rate_limits`.
