# Request 5.6 — Generate-Lesson UI

## Objective
Pro-gated dashboard CTA "Generate a lesson" that opens a form (target language picker, topic hint, card count). On submit: spinner + cost note; navigates to `/app/decks/:newDeckId/review` on success. Free-tier users see a friendly "Pro feature" prompt (no upgrade flow per DEBT-001 — just a "ask Justin" message).
