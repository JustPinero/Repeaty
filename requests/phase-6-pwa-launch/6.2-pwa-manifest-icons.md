# Request 6.2 — PWA Manifest + Peaty Icons

## Objective
Make Repeaty installable as a PWA. Generate the remaining Peaty illustrations needed for the manifest (192 + 512 + maskable + splash + sleepy/inactive pose), wire `apps/web/public/manifest.webmanifest`, and add the relevant `<link>` + `<meta>` tags to `apps/web/index.html`. iOS Safari install hint included since iOS has no `beforeinstallprompt`.

## Acceptance Criteria → Test Mapping

| Criterion                                                                       | Test                                                                       |
| ------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| `apps/web/public/manifest.webmanifest` exists with name, short_name, start_url, display, icons | smoke E2E + Lighthouse (6.5)                                          |
| 192 + 512 PNG icons + maskable variant present at `apps/web/public/peaty/peaty-icon-{192,512,maskable}.png` | filesystem + manifest entry checks                  |
| Splash JPG present (iOS install fallback)                                       | filesystem check                                                           |
| Index.html links the manifest + theme-color meta + apple-touch-icon             | smoke E2E asserts `<link rel="manifest">` present                          |
| iOS Safari users see an "Add to Home Screen" hint pill in the dashboard if not installed | unit test on the install-hint component                            |

## RED Phase
- New unit test for the install-hint component (`Dashboard.test.tsx` extension or new `InstallHint.test.tsx`).
- Smoke test asserts manifest.webmanifest fetch returns 200 + valid JSON with the right shape.

## GREEN Phase

1. **Generate the missing Peaty illustrations** per `assets/peaty/peaty-poses.md` (poses 9 sleepy + icon 192/512/maskable + splash). Save to `assets/peaty/` (source-of-truth) and `apps/web/public/peaty/` (served).
2. **Author `apps/web/public/manifest.webmanifest`:**
   ```json
   {
     "name": "Repeaty",
     "short_name": "Repeaty",
     "start_url": "/",
     "display": "standalone",
     "background_color": "#fff7e6",
     "theme_color": "#7bbf3a",
     "icons": [
       { "src": "/peaty/peaty-icon-192.png", "sizes": "192x192", "type": "image/png" },
       { "src": "/peaty/peaty-icon-512.png", "sizes": "512x512", "type": "image/png" },
       { "src": "/peaty/peaty-icon-maskable.png", "sizes": "512x512", "type": "image/png", "purpose": "maskable" }
     ]
   }
   ```
3. **Update `apps/web/index.html`** with the manifest link, theme-color meta, apple-touch-icon, and apple-mobile-web-app-capable meta.
4. **`InstallHint` component** for iOS Safari (no `beforeinstallprompt`) — small dismissable pill in the dashboard. Mounted only when `navigator.standalone === false` AND `/iPad|iPhone|iPod/.test(navigator.userAgent)`.
